# Keyboard Shortcuts

Quick reference for common actions in AinCore Notes.

## Editor

| Shortcut | Action |
|----------|--------|
| `Ctrl+B` | **Bold** text |
| `Ctrl+I` | *Italic* text |
| `Ctrl+K` | Insert link |
| `Ctrl+Shift+K` | Insert code block |
| `Ctrl+N` | New note |
| `Ctrl+S` | Save note |
| `Ctrl+F` | Search in current note |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+P` | Command palette |
| `Ctrl+E` | Toggle editor/preview |
| `Ctrl+\\` | Toggle sidebar |
| `Ctrl+1-9` | Switch to tab 1-9 |

## AI Features

| Shortcut | Action |
|----------|--------|
| `Ctrl+Space` | Trigger AI autocomplete |
| `Ctrl+Shift+S` | Summarize selected text |
| `Ctrl+Shift+G` | Grammar check |

## Tips

- Press `Ctrl+/` to see all available shortcuts
- AI features require a model loaded in AinCore
- You can customize shortcuts in Preferences → Keybindings
