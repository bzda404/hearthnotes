# Welcome to AinCore Notes

Welcome to your local-first knowledge management platform!

## What is AinCore Notes?

AinCore Notes is a **local AI-powered** markdown editor and knowledge base. All your data stays on your device — nothing is uploaded to any server.

## Key Features

- **Local AI**: Powered by AinCore, AI features run entirely on your machine
- **Markdown Editor**: WYSIWYG editing with live preview
- **BM25 Search**: Fast full-text search across all your notes
- **MCP Protocol**: Connect to external AI tools when needed
- **Privacy First**: Built-in PII detection and consent management

## Getting Started

1. Create a new note using `Ctrl+N` (or `Cmd+N` on Mac)
2. Use the AI panel on the right to get suggestions
3. Organize your notes into folders using the sidebar

> **Tip**: The AI features require AinCore to be running. Check the status indicator in the top bar.
